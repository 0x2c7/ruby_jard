| Command | Default key binding | Alias |
| ------- | ------------------- | ----- |
| `jard show [screen]` | | |

Show a particular screen on the current interface. This option correlates to recent layout's screens. If a screen is not available in current layout, it won't be shown.

**Examples:**

```
jard show source
jard show variables
jard show threads
jard show backtrace
```
