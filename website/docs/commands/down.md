| Command | Default key binding | Alias |
| ------- | ------------------- | ----- |
| `down` | Shift+F6 |  |

Explore the lower frame. See [up command](/docs/commands/up) for more information.

**Examples:**

```
down     # Move to lower frame
down 3   # Move to lower 3 frames
```
